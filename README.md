---

## Deploy to Vercel (fastest)

1. Create GitHub repo → upload these 3 files
2. Go to vercel.com → "Add New Project" → connect repo → Deploy
3. Copy your Vercel URL (e.g. `https://cryptowolf-bot.vercel.app`)

---

## After deploy — update YOUR_URL

Open `index.html` and replace ALL instances of `YOUR_URL` with your actual URL:

```
YOUR_URL → https://cryptowolf-bot.vercel.app
```

There are 4 places to update (all in the <head> meta tags).

Commit and push → Vercel auto-redeploys.

## Deploy to GitHub Pages (alternative, free)

1. Create repo named: `cryptowolf-bot`
2. Upload files
3. Settings → Pages → Branch: main → Save
4. URL: `https://YOURUSERNAME.github.io/cryptowolf-bot`
